# Day2-New-Repo
Recipe for Creating a New Working Environment:

1. Create a new repo or fork an existing one so that you are the owner of the new git repo.
2. Click the green code button on the repo's home page, and then click create new codespace.
   a. In the codespace that will open in your computer's browser, find the three lines and select open in VS Code Desktop. This allows you to work locally.
3. Once you are in VS Code Desktop, open a virtual environment and install required packages.
4. Freeze dependencies into a requirements.txt file.
5. Now you can write code as needed into .py files.
6. Git add your changes, commit them, and the push to github.
